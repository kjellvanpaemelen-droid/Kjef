Hypertext Pre-Processor PHP

Samenvatting van de les:

In de laatste sessie van Project ICT maakten we kennis met PHP, een veelgebruikte programmeertaal voor het bouwen van interactieve websites. We ontdekten hoe PHP werkt en gingen zelf aan de slag met verschillende oefeningen via een lokale server om de mogelijkheden van deze taal te verkennen, zoals het verwerken van formulieren.
In het tweede deel van de les werkten we aan het ontwikkelen van onze website. De focus lag op het integreren van PHP via een formulier. Omdat de campuscomputers een lokale serveromgeving kunnen opstarten, konden we onze PHPbestanden correct laten interpreteren. Op die manier konden we testen of ons formulier er goed uitzag, of de ingevoerde gegevens juist verwerkt werden en of de browser de verwachte output teruggaf.

Werking:

Hypertext Pre-Processor is een side-server programmeertaal die structureel aardig wat weg heeft van C. Deze taal werkt via een server in plaats van lokaal via een browser. De werking ziet er als volgt uit:

  1. De browser haalt het PHP-bestand op
  2. Via de server gaat dit naar een PHP-interpreter
  3. Het bestand wordt uitgevoerd en omgevormd naar een HTML
  4. De server verstuurt deze HTML naar de browser en wordt zichtbaar voor de gebruiker

Belangrijke commands:

\": dubbel aanhalingsteken
\n: new line
\t: tab
$: dollarteken
\\: backslash
//: commentaar
